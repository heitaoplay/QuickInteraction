# XSAct-QuickAction v1.1.1 发布说明

**发布日期**：2026-07-21
**类型**：底层重构（无新功能，建议更新以获得更稳定的体验）

## 本次变更

本次为纯底层架构优化，运行表现与 v1.1.0 完全一致，所有已验证的功能（我的动作列表交互重设计、工具栏图标化、书签失效修复等）均不受影响。

### 重构内容
- **单文件拆分**：将原先 5179 行的单文件 `quick-interaction.user.js` 拆分为 `src/` 目录下 **22 个模块化文件**（逻辑模块均 ≤350 行，仅大 CSS 块作为样式资源例外）。构建脚本 `build.mjs` 改为按序拼接 `src/` 生成发布产物，符合团队代码规范 §2/§10。
- **删除死代码**：移除 `getRoomCharacters` 的重复定义（仅保留受 `selfMode` 控制的正确版本），消除同名函数覆盖带来的语义隐患。
- **收口静默失败**：将 10 处空 `catch` 统一改为可见的 `console.warn`，消除「静默吞错」红线隐患（涉及存储读写、自定义动作注册/检测、拖拽数据、更新检查等）。

### 验证结果
- `node --check` 语法校验通过
- 构建产物约 307KB，与旧版体积基本持平
- 产物中 `getRoomCharacters` 仅 1 处定义、无残留空 catch、关键函数齐全

## 升级提示
- **油猴用户**：刷新 Bondage Club 页面即自动更新；或在脚本端 5 分钟轮询周期内收到更新/公告横幅。
- **书签 / 控制台用户**：若此前按 v1.1.0 重建过书签，本次无需变动；如未创建，请参照 README「书签安装」章节重新创建（已带加载反馈与前提说明）。
- **重复加载提醒**：若同时装过完整版 `quick-interaction.user.js`，建议先在油猴里删除其一，避免重复初始化。
